## üöÄ How to Use This Repository

1. Clone the repository:

```bash
git clone https://github.com/hv7275/60-Days-Python-Series.git
cd 60-Days-Python-Series
```

2. Install Python (3.8+ recommended):

- Windows: download from python.org and enable "Add Python to PATH".
- macOS / Linux: use system package manager or pyenv.

3. Install Jupyter Notebook (or use VS Code's Jupyter support):

```bash
python -m pip install --upgrade pip
pip install jupyter
```

4. Open the notebook for the day and run the cells. In the repo root you can run:

```powershell
jupyter notebook
```

Or open the file directly in VS Code (with the Python and Jupyter extensions) and run the cells inline.
